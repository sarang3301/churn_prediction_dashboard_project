Customer Churn Prediction Dashboard
Project Overview
This project predicts whether a customer will churn (leave a service) using Machine Learning.  
It also provides an interactive dashboard built with Streamlit.

---

Problem Statement
Customer churn is a major problem in telecom and other industries.  
This project helps businesses identify customers likely to leave.

---


Tech Stack
- Python 🐍
- Pandas
- Scikit-learn
- Streamlit
- Machine Learning
